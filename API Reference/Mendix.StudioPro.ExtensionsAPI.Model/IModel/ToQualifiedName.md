# IModel.ToQualifiedName&lt;T&gt; method

Create an object reference from its string representation.

```csharp
public IQualifiedName<T> ToQualifiedName<T>(string fullName)
    where T : class
```

| parameter | description |
| --- | --- |
| T | Type of object referenced by [`IQualifiedName`](../IQualifiedName-1.md). |
| fullName | String representation of [`IQualifiedName`](../IQualifiedName-1.md). |

## Return Value

Created [`IQualifiedName`](../IQualifiedName-1.md) which [`FullName`](../IQualifiedName/FullName.md) equals *fullName*. Note, that the reference might be invalid, and hence its [`Resolve`](../IQualifiedName-1/Resolve.md) method might return `null`.

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | *T* is not a type from ExtensionsAPI assembly. |
| ArgumentOutOfRangeException | *T* cannot be referenced. |

## Remarks

Note, that this method returns a value even when the object reference is invalid. That is when no object of type *T* exists in the app with the QualifiedName matching *fullName*. In that case [`Resolve`](../IQualifiedName-1/Resolve.md) of the result would return `null`.
