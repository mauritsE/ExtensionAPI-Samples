# IDocument interface

Representation of a single [document](https://docs.mendix.com/apidocs-mxsdk/mxsdk/projects-metamodel#2-2-documents) in metamodel. This is an abstract unit, so all instances of [`IDocument`](./IDocument.md) also implement more concrete interface like [`IMicroflow`](../Mendix.StudioPro.ExtensionsAPI.Model.Microflows/IMicroflow.md).
