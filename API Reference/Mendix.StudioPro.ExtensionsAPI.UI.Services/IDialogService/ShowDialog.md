# IDialogService.ShowDialog method

Shows a modal dialog based on the passed [`ModalDialogViewModelBase`](../../Mendix.StudioPro.ExtensionsAPI.UI.Dialogs/ModalDialogViewModelBase.md). The [`ModalDialogViewModelBase`](../../Mendix.StudioPro.ExtensionsAPI.UI.Dialogs/ModalDialogViewModelBase.md) dialog is currently only allowed to be created by implementing the abstract class [`WebViewModalDialogViewModel`](../../Mendix.StudioPro.ExtensionsAPI.UI.Dialogs/WebViewModalDialogViewModel.md), which will allow using a [`IWebView`](../../Mendix.StudioPro.ExtensionsAPI.UI.WebView/IWebView.md) inside a modal dialog.
