# IMessageBoxService.ShowQuestion method

Shows a message box with a question icon and the question provided.

```csharp
public string ShowQuestion(string question, string? details = null, string[]? buttons = null, 
    string? defaultButton = null)
```

| parameter | description |
| --- | --- |
| question | the question to ask the user. |
| details | optional extra information, which will be shown after the user clicks "Show Details" button. |
| buttons | optional list of buttons instead of the default 'Close' button. |
| defaultButton | optional button with keyboard focus when the dialog opens. It must be from the supplied list of buttons. |

## Return Value

the text of the button which was clicked, as the answer to the question. If user did not provide custom buttons, it will be 'Close'.
