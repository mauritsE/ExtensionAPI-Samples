# ISelectorDialogService.SelectDocumentAsync&lt;TDocument&gt; method

Shows interface (in a popup) for selecting [`IDocument`](../../Mendix.StudioPro.ExtensionsAPI.Model.Projects/IDocument.md) of a specific type *TDocument* from all such documents in the app.

```csharp
public Task<SelectorResult<TDocument>> SelectDocumentAsync<TDocument>(
    DocumentSelectorDialogOptions<TDocument> options)
    where TDocument : class, IDocument
```

| parameter | description |
| --- | --- |
| TDocument | The type of the document that can be selected. It must be a concrete type like [`IMicroflow`](../../Mendix.StudioPro.ExtensionsAPI.Model.Microflows/IMicroflow.md) and not a base type like [`IMicroflowBase`](../../Mendix.StudioPro.ExtensionsAPI.Model.Microflows/IMicroflowBase.md). If you need to select a document of a one of a few types, use [`SelectDocumentUntypedAsync`](./SelectDocumentUntypedAsync.md). |
| options | An instance of [`DocumentSelectorDialogOptions`](../DocumentSelectorDialogOptions-1.md) that describes how the selector dialog should behave. |

## Return Value

A Task that is resolved when the selection process is finished by Studio Pro user. The Result contains an instance of [`SelectorResult`](../SelectorResult-1.md) representing the result of the selection process.

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentNullException | *options* is null. |
| ArgumentException | *TDocument* does not refer to a concrete type, but to a base type. |

## Remarks

Use [`SelectDocumentUntypedAsync`](./SelectDocumentUntypedAsync.md) if you need to select a document of one of a few types, e.g. select either Microflow or Nanoflow.
