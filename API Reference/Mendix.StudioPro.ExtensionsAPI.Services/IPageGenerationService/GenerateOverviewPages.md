# IPageGenerationService.GenerateOverviewPages method

Generate edit and summary pages for the specified entities of a module.

```csharp
public IEnumerable<IPage> GenerateOverviewPages(IModule module, IEnumerable<IEntity> entities, 
    bool generateIndexSnippet = false)
```

| parameter | description |
| --- | --- |
| module | Module in which the OverviewPages folder and the pages themselves will be generated. |
| entities | Entities based upon which the overview pages will be generated. |
| generateIndexSnippet | Specifies whether an index snippet for the entities should be generated. An index snippet is only generated if more than one entity is passed into *entities* |

## Remarks

This API requires at least one layout of type Responsive and one layout of type Popup or ModalPopup to be present in the App model. These layouts will be used for the generated pages.
